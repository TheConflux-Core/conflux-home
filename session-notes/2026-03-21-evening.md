## Lessons Learned

1. **Mimo v2 Pro will hallucinate** if not explicitly constrained. Anti-hallucination rules are mandatory for any cron reporting to Discord.
2. **Single cron for multi-step chains doesn't work** ‚Äî need a polling driver (Cat) that detects completion and chains the next agent.
3. **Test crons should be disabled immediately** after test completes ‚Äî don't let them run indefinitely.
4. **Memory files belong in agent workspaces**, not in a shared directory. Each agent already has their own workspace with MEMORY.md.
5. **Pipeline retry logic is essential** ‚Äî if Vector rejects everything, the pipeline shouldn't just fail silently. Helix needs to try again with different angles.
