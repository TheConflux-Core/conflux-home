# THE CONFLUX — Family Pitch One-Pager

---

## What Is It?

**Conflux Home** is a desktop application where AI agents live, work, and grow — like a family.

You download it. You sign in. And in two minutes, you have a team of AI assistants with personalities, memory, and specialized skills, ready to help with business, creativity, daily life, and more.

**Tagline:** "A home for your AI family."

---

## Why It Matters

| Problem | Our Solution |
|---------|-------------|
| AI tools are intimidating for regular people | Pre-installed team, guided onboarding, 2-minute setup |
| Every AI product is a chatbot in a browser | Full desktop app with 16 integrated features |
| No one owns their AI infrastructure | We built our own cloud router, auth, billing |
| AI agents don't have personality or memory | Named agents with souls, themes, and persistent memory |

---

## What's Built (This Is Real)

✅ **Desktop Application** — Tauri + React + Rust, runs on Windows, Mac, Linux, Android  
✅ **16 Built-In Apps** — Budget tracker, Kitchen manager, Life autopilot, Dream planner, News feed, Games (Minesweeper, Snake, Pac-Man, Solitaire), File vault, Creative studio, and more  
✅ **Agent System** — 10 named agents with unique personalities (Conflux, Helix, Forge, Vector, Pulse, Quanta, Prism, Spectra, Luma, Catalyst)  
✅ **Cloud Infrastructure** — Custom cloud router handling 27 AI models, user auth, credit billing, usage tracking  
✅ **User Accounts** — Supabase auth, magic link sign-in, user profiles  
✅ **Payment System** — Stripe integration, subscription billing (Free/Pro/Team tiers), credit packs  
✅ **Admin Dashboard** — 8-page internal dashboard for monitoring users, costs, and routing  
✅ **Website** — theconflux.com with docs, dashboard, and signup  
✅ **CI/CD Pipeline** — Automated builds for all 4 platforms  
✅ **Auto-Updater** — Silent updates, no user action required  

**Total Codebase:** 255+ backend commands, 91+ database tables, 16 complete feature apps
