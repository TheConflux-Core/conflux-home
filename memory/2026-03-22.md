## Lessons Learned

1. **Build velocity compounds** — Once the SDK was built, every subsequent sprint was faster because they all consumed the same API client.
2. **Parallel subagents are OP** — Running 2-3 Forge instances simultaneously cut sprint time in half.
3. **Polish matters more than features** — The 10 polish fixes made the app feel alive vs. functional.
4. **Tauri v2 has learning curve** — Config format, permissions, capabilities all changed from v1. Need to read actual error messages, not guess.
5. **DALL-E 3 is dirt cheap** — 22 custom images for $0.88. No reason not to have custom avatars.
6. **Don builds at 2 AM** — Peak productivity window. Don't suggest sleeping during a build session.
