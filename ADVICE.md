# Agent Team Expansion — Strategic Advice (March 26, 2026)

## Context
After GitHub API key leak, discussing security, legal, and operational gaps in the agent team.

## Current Team (Building)
Vector, Conflux (ZigBot), Prism, Spectra, Luma, Helix, Forge, Quanta, Pulse — all focused on building. Zero agents focused on protecting.

## Recommended New Agents

### 1. Security — Immediate Priority

**Sentinel (Red Team)** — Offensive security, vulnerability research, pen testing. Reviews code before it ships. Audits infrastructure. Finds holes before attackers do.

**Aegis (Blue Team)** — Defensive security, monitoring, incident response, hardening. Watches logs, detects anomalies, manages access controls.

**Why now:** Building in the open. Every commit is public. Every vulnerability visible. Next leak could be user data, signing keys, or business logic.

### 2. Legal/IP — High Priority

**Counsel** — Legal research, document drafting, compliance monitoring. Contracts, patents, trademark infringement monitoring, tax research.

**Honest truth:** AI agent cannot replace a lawyer. Counsel handles 80% of prep work, real attorney handles 20% requiring a license.

**Entity:** LLC or corporation before accepting money. CPA and attorney should finalize.

### 3. Financial Operations

**Ledger** — Bookkeeping, revenue tracking, cost analysis, tax prep. Watching the money especially once API proxy handles real revenue.

### 4. Data & Privacy

**Custodian** — GDPR/CCPA compliance, data handling, user privacy.

## Priority Order
1. **This week:** Sentinel + Aegis. Security review of everything built so far.
2. **Before accepting money:** Counsel. Draft ToS, privacy policy, review entity structure.
3. **Before scaling users:** Ledger. Revenue tracking, cost monitoring, tax prep.
4. **Before marketing push:** Custodian. Data handling compliance.

## CI/CD Integration
Don't need separate agent. Need security gates in CI: Build → Test → Security Review (Sentinel) → Merge. GitGuardian already caught the API key leak.

## Big Picture
The empire grows on what you build. It survives on what you protect.
